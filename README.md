# Students_Grade_Predictor
Predicts a students' 3rd semester grades based on their 1st & 2nd semester grades, time spent studying, instances of failures, and number of absences.

Uses a linear regression model

Written in Python

System Requirements: Anaconda, TensorFlow, Pandas, NumPy, scikit-learn, pickle, matplotlib

Data Source: https://archive.ics.uci.edu/ml/datasets/student+performance
